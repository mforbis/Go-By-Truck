sendErrorEmail: 
Adds capability to send an email to errors@gobytruck.com when an error occurs in the mobile application.
Input Parameters
sid (optional) - user guid for the person logged into the mobile application recoverable (required) - Let's GBT Web Developers know the nature of the error. recoverable=true or recoverable=false recoverable means the operation can resume normally but there may be anomalies. Unrecoverable means the mobile application ran into a problem and normal operation failed. subject (optional) - subject of the email (I will pre-pend "GBT Mobile Error - " to the subject sent.) body (required) - body if the email being sent (all emails will be sent as HTML - so add markup as you wish)
Expected Output
// REQUEST (Missing Body): https://localhost/gbtspring/mobile/sendErrorEmail?sid=1330&subject=drop%20down%20error&body=&recoverable=true // RESPONSE: {"status":"false","error_msg":{"errorMessage":"INSUFFICIENT_PARAMETERS","error":"Email Body Missing"}} // REQUEST (Missing Recoverable Flag): https://localhost/gbtspring/mobile/sendErrorEmail?sid=1330&subject=drop%20down%20error&body=Error%20while%20trying%20to%20build%20trailer%20type%20drop%20down.%20List%20value%203%20is%20empty.&recoverable= // RESPONSE: {"status":"false","error_msg":{"errorMessage":"INSUFFICIENT_PARAMETERS","error":"recoverable (variable) must equal true or false"}} // REQUEST (Valid Parameters): https://localhost/gbtspring/mobile/sendErrorEmail?sid=1330&subject=drop%20down%20error&body=Error%20while%20trying%20to%20build%20trailer%20type%20drop%20down.%20List%20value%203%20is%20empty.&recoverable=true // RESPONSE: {"status":"true"}
Possible Errors
INSUFFICIENT_PARAMETERS message = Email Body Missing INSUFFICIENT_PARAMETERS message = "recoverable" must equal true or false
